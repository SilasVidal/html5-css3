Negrito & Itálico

Esse é um assunto muito importante,pois além de dar mais sentido as coisas,deixa tudo mais claro para nós.

Sem contar que algumas tags se tornaram obsoletas,e algumas até foram apagadas

E a tag <em> significa que queremos dar ênfase ao termo. Logo aparecerá em itálico.

Marca Texto

As marcações no texto servem para dar ênfase em uma parte do texto,o qual se trata de um trecho muito valioso ←como esse.

Para fazer essa marcação no html usamos a tag <mark>.

E para fazer um texto com letras pequnas basta usar a tag <small>.

Texto Deletado

Antigamente existia a tag <strike>,mais está depreciada pela última versão do HTML5.No lugar dela entrou a tag <del>,que significa que o texto está ali,pode ser lido,mais deve ser desconsiderado pelo leitor.Mais não desconsidere nesse caso 😂

Texto Inserido

O texto inserido é o exato oposto do texto deletado.Nesse caso,se colocarmos um texto qualquer dentro de <ins> e </ins>,estamos dizendo que o texto está ali,deve ser lido e você deve prestar atenção nele.

Texto sobrescrito e subscrito

Para escrever com letras minúsculas abaixo ou acima de uma palavra,usamos <sub> e <sup>,sub para baixo,e sup para cima.
